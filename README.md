# Vaultify: PSEI & Crypto Portfolio Tracker

A high-precision, privacy-focused investment tracker for Philippine Stocks (PSEI) and Cryptocurrency. Built as a Single Page Application (SPA) ready for GitHub Pages.

## Features
- **Multi-Asset Support**: Track stocks (e.g., COL Financial), Crypto (e.g., Coins.ph, Bybit), and Bank Savings.
- **WAC Calculation**: Weighted Average Costing excluding transaction fees from capital for pure cost-basis tracking.
- **High Precision**: 4-decimal place tracking for prices and quantities.
- **AI Insights**: Real-time market sentiment and news powered by Google Gemini.
- **Cloud Sync**: Securely backup and restore your data across devices using unique Sync Codes (via JSONBin.io).
- **Platform Management**: Dynamic management of your investment platforms.

## Tech Stack
- React 19
- Tailwind CSS
- Google Gemini API
- Recharts (Visualizations)
- ESM Modules (No build step required for modern browsers)

## Deployment
This app is designed to be hosted on GitHub Pages. Simply upload the root files to a repository and enable GitHub Pages in settings.
